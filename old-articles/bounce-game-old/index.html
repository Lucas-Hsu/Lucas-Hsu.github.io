Bounce Game PyGame Version

Features

The PyGame version of Bounce Game is called pygame_BOUNCE.py. The background, ball, and orb sprites can all be replaced by any image renamed to theBackground.png, blue_square.png, and theBlock.png respectively. There was also a GUI launcher that allowed you to modify many of the game’s parameters.

Still, many of the things were quite simplistic. The score is simply outputted using a print statement. Looking at my old highscore file, it seems my BOUNCE-ing skills have diminished.

Still, due to it being a Python game, I had more low-level control over things. The coolest thing was definitely the toggleable ghosting effect achieved by not completely painting over the canvas each frame.

I’ve cleaned things up a bit and decided to upload the PyGame version too, as it contains several features that the Godot version did not implement. Use the PyGame BOUNCE Download Link to get the script and game files for pygame_BOUNCE_release.py.
