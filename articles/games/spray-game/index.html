Spray Game

In this game, you are an elephant in a water balloon fight against monkeys!

I spent 40 hours in December 2023 to learn how to use Unity and make everything in this game. I also created a manual with a few tips to help new players learn how to play this difficult game.

I became quite skilled at Spray Game playing it during development, so I continued increasing its difficulty to match my skill level. It became too difficult for beginners, so I added a Baby Mode for beginner players to try out. In January 2026, I re-built the game make the difficulty setting a simple toggle button and updated the manual accordingly.
