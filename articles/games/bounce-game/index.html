Bounce Game

You propel a rolling smiley face to collect shiny orbs.

Click to move, Escape to quit.

The force and direction the ball gets propelled is based on the distance between the mouse cursor and the ball. Move faster when hitting the orange orbs to get a higher score!

I first made this game in Scratch, then remade it using PyGame when I learned Python. However, PyGame was difficult to use. The result was even uglier than the version made in Scratch. When I heard that GDScript in Godot Engine is very similar to Python, I remade the remake in Godot. I think it looks great now.

I made everything in this game except the normal map for the smiley face.
