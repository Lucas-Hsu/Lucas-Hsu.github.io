Bounce Game

You propel a rolling smiley face to collect shiny orbs.

Click to move, Escape to quit.

The force and direction the ball gets propelled is based on the distance between the mouse cursor and the ball.

Move faster when hitting the orange orbs to get a higher score!

Play The Game

You can download the game on the Bounce Game Github Releases page.

You can also play Bounce Game in your browser via Itch.io.

Development History

  1. Scratch version: The base game with a simple scoring system and basic sound effects. Created in basically one afternoon.
  2. PyGame version: Unappealing graphics, but had a cool toggleable ghosting effect via not completely refreshing the canvas. Added right-click to freeze ball in place for more precise control.
  3. Godot version: Added a nice color palette, basic lighting, and more sound effects. Scoring system improved to incentivize higher speeds.

Color Palette

I created a color palette, which I call NightLight, around the time I remade BounceGame in Godot. I only used the blue and orange hues in the game, as I didn’t need more colors. However, I’ve used the other colors in some of my diagrams, so they might seem familiar. I like how the saturated colors contrast with the dark blue.
