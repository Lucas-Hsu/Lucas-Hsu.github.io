Making My Personal Website

[2026 Jan. 22]

This is a complete remake of my personal website. I was dissatisfied with the slow loading times due to all the javascript and background images. Also, the nested loops hierarchy was way too complicated.

Now, I’m using a website builder called Astro, which builds my pages before I deploy, so that these pages contain no javascript and are optimized to be as quick to load as possible. Also, most pages are defined from a one of my templates, so new blog posts or articles only need to be written in markdown.
